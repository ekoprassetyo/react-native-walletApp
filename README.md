## Run Locally

Clone the project

```bash
  git clone https://github.com/ekoprassetyo/react-native-walletApp.git
```

Go to the project directory

```bash
  cd react-native-walletApp
```

Install dependencies

```bash
  npm install
```

Start android

```bash
  npx react-native run-android
```
